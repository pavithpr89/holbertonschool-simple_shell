.SH NAME
_printf \- The "Holberton special" for formatted output conversion
.SH SYNOPSIS
.B int _printf(const char *format, ...);
.SH DESCRIPTION
The _printf function produces output according to a format string. 

.TP
.B It currently supports the following conversions:
%c - Prints out a single character

%s - Prints out a string of text

%d - Prints out a decimal in base 10

%i - Prints out an integer in base 10

%u - Prints out an unsigned decimal integer

%o - Prints out a decimal in base 8

%x - Prints out a decimal in base 16 ( Lowercase ) 

%X - Prints out a decimal in base 16 ( Uppercase )

%% - Prints a % sign

%b - Prints an unsigned int (any positive number) as binary

%S - Prints the string of text using printable ascii characters.
     Anything that isn't a printable ascii character will instead print a 
     hexadecimal code for 2 digits (\\xXX)

.SH IMPLEMENTATION
.TP
The way to use this function, basically works exactly the same way you would use printf().
.RS

_printf("I have some text here and a number: %d", num);
.RE

Standard formatting (new lines, backslashes etc) still work as per the original printf() 


.SH RETURN VALUE
Returns the number of characters printed, or -1 on error.
